System Settings

The Settings Module allows administrators to configure system-wide operational settings. These settings are used by different modules throughout the system to calculate attendance, working hours, payroll, and employee performance.

Attendance Settings

Attendance settings define the total working days and expected office hours for a month. These values are used to calculate employee attendance percentages, working hours, overtime, payroll, and performance statistics.

  • ✓ Configure Monthly Working Days
  • ✓ Configure Monthly Office Hours
  • ✓ Update Attendance Rules

Example Configuration

  • Working Days: 26 Days
  • Office Hours: 234 Hours

Attendance Settings Workflow

Open Settings → Select Attendance → Update Working Days → Update Office Hours → Save Configuration → System Calculations Updated
