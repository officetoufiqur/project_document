Payroll Management Module

The Payroll Management Module automates salary processing based on employee attendance records, bonus calculations, payroll approvals, and payslip generation. This module ensures accurate salary management and provides employees access to their approved payslips.

1. Payroll List

Payroll is automatically generated based on the previous month's attendance records.

  • ✓ Generate Monthly Payroll
  • ✓ Attendance Based Salary Calculation
  • ✓ Bonus Generation
  • ✓ Payroll Edit
  • ✓ Payroll Delete
  • ✓ Payroll Approval
  • ✓ Auto Payslip Generation

Bonus Policy

When the "Generate Bonus" button is clicked, the system automatically calculates 50% of Basic Salary and adds it to payroll for employees whose bonus_eligibility = true.

2. Payslip Management

Approved payroll records are automatically converted into employee payslips.

  • ✓ View Payslip
  • ✓ Download Payslip PDF
  • ✓ Employee Self Service
  • ✓ Monthly Payslip History
  • ✓ Salary Breakdown
  • ✓ Bonus Breakdown
  • ✓ Deduction Breakdown

Payroll Processing Workflow

Previous Month Attendance → Generate Payroll → Salary Calculation
Generate Bonus → Bonus Added → Payroll Review
Approve Payroll → Payslip Generated → Employee Can View & Download

Key Features
