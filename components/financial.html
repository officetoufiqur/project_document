Financial Management Module

The Financial Management Module helps manage company accounts, expenses, vouchers, payments, and financial transactions. It ensures proper expense approval workflows, account balance tracking, and payment processing.

1. Accounts Management

Manage all financial accounts with opening balances and account classifications.

  • ✓ Create Account
  • ✓ Edit Account
  • ✓ Delete Account
  • ✓ Opening Balance Management
  • ✓ Account Balance Tracking

Account Types

  • • Assets
  • • Salary
  • • Liability
  • • Equity
  • • Revenue
  • • Expense

2. Expense Categories

Manage expense categories used throughout the financial system.

  • ✓ Create Category
  • ✓ Update Category
  • ✓ Delete Category
  • ✓ Expense Classification
  • ✓ Category Management

3. Expenses Management

Manage all company expenses with account and category assignment.

  • ✓ Create Expense
  • ✓ Assign Expense Account
  • ✓ Assign Expense Category
  • ✓ Edit Expense
  • ✓ Delete Expense
  • ✓ View Expense Details
Every newly created expense is automatically added to the Expense Logs section for review and approval.

4. Expense Logs

Review and approve submitted expenses before payment.

  • ✓ Expense Log List
  • ✓ View Expense Details
  • ✓ Approval Workflow
  • ✓ Status Tracking
  • ✓ Expense Verification
Approved expenses are automatically transferred to the Vouchers section.

5. Voucher Management

Manage approved expenses and process payments.

  • ✓ Approved Expense List
  • ✓ Voucher Status Tracking
  • ✓ View Voucher Details
  • ✓ Full Payment Processing
  • ✓ Partial Payment Processing
  • ✓ Select Payment Account
  • ✓ Payment History
  • ✓ Invoice Generation

Payment Process

  • 1. Select Voucher
  • 2. Choose Payment Type (Full / Partial)
  • 3. Select Payment Account
  • 4. Enter Payment Amount
  • 5. Account Balance Automatically Deducted
  • 6. Payment Invoice Generated

6. Payment Invoices

View and manage all completed voucher payment invoices.

  • ✓ Payment Invoice List
  • ✓ View Invoice
  • ✓ Download Invoice PDF
  • ✓ Payment History
  • ✓ Voucher Reference Tracking
  • ✓ Account Transaction Reference

Financial Management Workflow

Create Expense → Expense Logs → Review Expense → Approve Expense → Voucher Created → Process Payment → Deduct Account Balance → Generate Invoice
