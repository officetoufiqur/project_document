Recruitment & Onboarding Module

The Recruitment & Onboarding Module manages the complete hiring process, from job requisition creation to employee onboarding. This module helps HR teams streamline recruitment workflows, candidate management, interview scheduling, offer generation and employee joining processes.

1. Careers Management

Administrators can dynamically manage the company careers page.

  • ✓ Create Career Page Content
  • ✓ Update Career Information
  • ✓ Publish Job Opportunities
  • ✓ Manage Company Career Listings

2. Job Requisitions

HR and department heads can create recruitment requests.

  • ✓ Create Requisition
  • ✓ Edit Requisition
  • ✓ Delete Requisition
  • ✓ Approve / Reject Requisition

3. Job Management

Approved requisitions are converted into job postings.

  • ✓ Create Job Templates
  • ✓ Publish Job Posts
  • ✓ Update Job Information
  • ✓ Delete Job Posts
  • ✓ Display Jobs on Careers Page

4. Job Applications

Manage all candidate applications submitted through the careers page.

  • ✓ View Applicant List
  • ✓ Download Candidate Resume
  • ✓ Review Candidate Profiles
  • ✓ Track Application Status

5. Interview Scheduling

Schedule and manage interviews for shortlisted candidates.

  • ✓ Create Interview Schedule
  • ✓ Update Interview Information
  • ✓ View Scheduled Interviews
  • ✓ Download Candidate Resume
  • ✓ Manage Date & Time Slots
  • ✓ Candidate Tracking

6. Offer Generation

Generate offer letters for selected candidates.

  • ✓ Generate Offer Letter
  • ✓ Define Salary Structure
  • ✓ Set Joining Date
  • ✓ Generate PDF Offer Letter
  • ✓ Candidate Offer Tracking

7. Employee Onboarding

Final onboarding process for candidates who accept the offer letter.

  • ✓ Verify Candidate Information
  • ✓ Complete Joining Process
  • ✓ Create Employee Record
  • ✓ Assign Department
  • ✓ Assign Designation
  • ✓ Generate Employee Account
  • ✓ Activate Employee Profile
  • ✓ Mark Onboarding as Complete
After successful onboarding, the candidate is converted into a company employee and receives access to the system based on assigned roles and permissions.
